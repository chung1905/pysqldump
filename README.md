# Pysqldump

Parallel mysqldump in Python

## Getting started

Can phai co mysqldump tren may
Viet file `.env` theo mau~
```shell
db_host=xxx
db_user=xxx
db_password=xxx
db_name=xxx
```

```shell
souce .env
python -m venv venv
source ./venv/bin/activate
pip3 install -r requirements.txt
python3 pysqldump.py
```
